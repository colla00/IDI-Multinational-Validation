# Data Access Instructions

## MIMIC-IV (v2.2) — Derivation Cohort
- URL: https://physionet.org/content/mimiciv/2.2/
- Requires: PhysioNet credentialed account + CITI "Data or Specimens Only Research" training
- After approval, download and place in: `data/raw/mimic-iv/`
- Key tables used: `admissions`, `icustays`, `chartevents`, `labevents`, `diagnoses_icd`

## HiRID (v1.1.1) — External Validation Cohort
- URL: https://physionet.org/content/hirid/1.1.1/
- Requires: PhysioNet credentialed account + HiRID-specific data use agreement
- After approval, download and place in: `data/raw/hirid/`
- Key tables used: `observation_tables`, `pharma_records`, `general_table`

## Directory Structure After Download
```
data/
├── raw/
│   ├── mimic-iv/
│   │   ├── hosp/
│   │   └── icu/
│   └── hirid/
│       ├── observation_tables/
│       └── pharma_records/
└── processed/         # Generated by src/cohort_selection.py
    ├── mimic_cohort.csv
    ├── hirid_cohort.csv
    └── idi_features.csv
```

**IMPORTANT: Never commit raw or processed patient data to this repository.**
All data files are excluded by .gitignore.
